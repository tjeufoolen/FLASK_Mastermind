# PYTHON-Mastermind
Assessment python blok 7

## Prerequisites
- virtualenvironment (venv)
- pip packages
    - flask
    
## Development tips
Automatic rebuilding on code change:
```
export FLASK_APP=main.py
export FLASK_ENV=development
flask run
```
